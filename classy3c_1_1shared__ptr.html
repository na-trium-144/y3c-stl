Detailed Description

template<typename T>
class y3c::shared_ptr< T >
  • operator* と operator-> 使用時にnullptrチェックを行う。
  • get() が返すポインタはラップされたものであり、 使用時にnullptrと参照先が生きているかのチェックができる。
  • 初期化は std::make_shared<T> または y3c::make_shared<T> で行う。
Examples
shared_ptr-null.cc, shared_ptr-ref.cc, and shared_ptr-unwrap-ref.cc.
