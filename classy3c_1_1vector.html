Detailed Description

template<typename T>
class y3c::vector< T >

可変長配列 (std::vector)

  • キャストするか unwrap() することで std::vector<T> (のconst参照)に戻せる。
    • std::vector<T> のconstでない参照で取得して変更を加えることはできないようにしている。
